Upload these files to a new GitHub repository and deploy on Vercel:
1. Create repo (public).
2. Upload all files from this folder (index.html, vecteezy_ogre.png, vecteezy_hammer.png).
3. On Vercel Dashboard click "New Project" -> import from GitHub -> choose repo -> Deploy.
4. After deploy, your site will be at https://your-repo-name.vercel.app
Don't forget to include attribution on the page: images from Vecteezy require credit.
